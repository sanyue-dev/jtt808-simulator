# Project Technology Stack

This document outlines the technology stack used in this project.

## Backend

- **Core Framework**: Spring Boot
- **Database**:
  - MySQL (mysql-connector-j)
  - MyBatis (mybatis-spring-boot-starter)
  - Spring Boot Starter JDBC
- **Web**:
  - Spring Boot Starter Web
  - Spring Boot Starter WebSocket
- **Template Engine**: FreeMarker (spring-boot-starter-freemarker)
- **Networking**: Netty (netty-all)
- **JSON Processing**: Gson
- **Protocol**: jtt808-protocol (Custom JT/T 808 protocol implementation)
- **Build Tool**: Maven
- **Java Version**: 17

## Frontend

- **Core Library**: jQuery (v2.1.1)
- **CSS Framework**: Bootstrap
- **Icon Fonts**: Font Awesome, Glyphicons, Climacons
- **Template Engine**: FreeMarker (for server-side rendering)
- **Maps**: Baidu Maps API
- **Charts**:
  - Flot
  - Morris.js
  - xCharts
  - Easy Pie Chart
  - JustGage
- **UI Components & Plugins**:
  - **Tables**: DataTables
  - **Forms**:
    - Chosen (Select boxes)
    - Select2 (Select boxes)
    - Autosize (Textareas)
    - Masked Input (Input masks)
    - Dropzone.js (File uploads)
    - Bootstrap Wizard (Wizards)
    - Bootstrap Editable (Inline editing)
  - **Calendar**: FullCalendar
  - **Notifications**: Toastr
  - **Other**:
    - Moment.js (Date handling)
    - BlockUI (UI blocking)
    - jQuery UI
    - jVectorMap (Maps)
